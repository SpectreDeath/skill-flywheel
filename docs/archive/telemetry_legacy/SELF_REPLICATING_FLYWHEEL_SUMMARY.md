```
Input: Usage logs + Ralph gold concepts
↓
1. Analyze execution patterns from last 10+ runs
2. Identify high-impact specialization opportunities
3. Cross-reference with Ralph insights
4. Select 3 specialized variants per popular skill
↓

```

↓
Output: 3 new specialized skills
