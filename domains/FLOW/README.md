# FLOW

Workflow configuration directory for Flywheel pipelines.

## Files

| File | Description |
|------|-------------|
| FLOW.full_cycle.yaml | Full cycle pipeline configuration |
| FLOW.onboard.yaml | Onboarding pipeline |
| FLOW.quality_audit.yaml | Quality audit pipeline |
| FLOW.skill_pipeline.yaml | Skill pipeline configuration |
| FLOW.stress_test.yaml | Stress testing pipeline |

## Note

This directory contains workflow configuration files (YAML), not SKILL.md files. It defines the Flywheel pipeline execution flows.
