# Formal Methods Skills

This directory contains a comprehensive collection of formal methods skills designed for AI agents and automated verification systems. These skills provide frameworks for mathematical rigor, theorem proving, model checking, and formal verification across various domains including TLA+, Coq, Isabelle/HOL, and hybrid verification approaches.

## Generated Skills

### 1. SKILL.tla_plus_specification.md
**Purpose**: TLA+ (Temporal Logic of Actions) specification systems for formal system modeling, concurrent system verification, and distributed algorithm specification.

**Key Capabilities**:
- State machine modeling using TLA+ for system behavior specification
- Temporal property specification using temporal logic
- Concurrent system verification with precise temporal reasoning
- Model checking integration with TLC (Temporal Logic of Actions Checker)
- Refinement and abstraction techniques for complex system modeling
- Invariant discovery and verification
- Performance optimization for efficient model checking

**Applications**: Distributed systems, concurrent algorithms, protocol verification, safety-critical systems

### 2. SKILL.coq_proof_engine.md
**Purpose**: Coq proof engines for interactive theorem proving, formal verification, and mathematical reasoning.

**Key Capabilities**:
- Proof automation using Coq's Ltac language and proof automation techniques
- Custom tactic development for domain-specific proof patterns
- Proof search strategies including backtracking, heuristics, and machine learning integration
- Dependent type theory for precise specification and verification
- Formal verification integration with software and hardware verification
- Proof optimization for performance, readability, and maintainability
- Interactive proof development with intelligent suggestions

**Applications**: Mathematical theorem proving, software verification, hardware verification, formal methods research

### 3. SKILL.isabelle_hol_theorem_proving.md
**Purpose**: Isabelle/HOL theorem proving frameworks for higher-order logic reasoning, formal verification, and mathematical theorem proving.

**Key Capabilities**:
- Higher-order logic reasoning using Isabelle's HOL for complex mathematical reasoning
- Isar structured proofs for readable, maintainable formal developments
- HOL-specific tactics and proof methods for HOL reasoning patterns
- Proof automation using Sledgehammer, metis, and custom tactics
- Formal verification integration with software and hardware verification workflows
- Mathematical library integration for analysis, algebra, and number theory
- Interactive theorem proving with intelligent proof state management

**Applications**: Mathematical research, formal verification, software correctness, academic research

### 4. SKILL.model_checking_frameworks.md
**Purpose**: Model checking frameworks for automated verification of finite-state systems, temporal logic model checking, and system property verification.

**Key Capabilities**:
- State space exploration algorithms for model checking
- Temporal logic verification (LTL, CTL, CTL*)
- Counterexample generation and analysis for failed properties
- Symbolic model checking using BDDs and SAT/SMT solvers
- Bounded model checking frameworks for bug detection
- Abstraction techniques and counterexample-guided abstraction refinement (CEGAR)
- Performance optimization through state compression and parallelization

**Applications**: Hardware verification, protocol verification, software model checking, safety analysis

### 5. SKILL.formal_verification_methods.md
**Purpose**: Formal verification methods for comprehensive system verification, including deductive verification, runtime verification, static analysis, and hybrid verification approaches.

**Key Capabilities**:
- Deductive verification using Hoare logic, separation logic, and program logics
- Runtime verification systems for monitoring and checking system properties during execution
- Static analysis techniques for detecting bugs, security vulnerabilities, and correctness issues
- Hybrid verification approaches combining multiple techniques for comprehensive coverage
- Verification integration with software development workflows and CI/CD pipelines
- Property-based testing frameworks for systematic testing with formal properties
- Verification automation with intelligent tool selection and configuration

**Applications**: Safety-critical systems, security verification, software quality assurance, compliance verification
