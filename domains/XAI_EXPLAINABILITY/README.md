# XAI & Explainability

Domain for explainable AI skills including SHAP, feature importance, and visualization.

## Skills

| Skill | Description |
|-------|-------------|
| [xai-plotly-visualization](xai-plotly-visualization/) | Interactive Plotly visualizations for XAI |
| [terminal-xai-reporting](terminal-xai-reporting/) | Terminal-friendly ASCII XAI reports |
| [shapiq-explainability-pipeline](shapiq-explainability-pipeline/) | SHAP-IQ for Shapley interaction indices |
| [shapiq-interaction-extraction](shapiq-interaction-extraction/) | Extract main effects and interactions |
| [global-feature-importance](global-feature-importance/) | Aggregate SHAP for global importance |

## Usage

Use these skills for:
- Feature importance analysis
- SHAP and SHAP-IQ computations
- Interaction effect analysis
- XAI visualizations
- Model interpretability
