## Purpose

Combine existing knowledge from multiple sources to form coherent solutions and new insights.

## Description

Knowledge synthesis involves gathering information from diverse sources, identifying connections and conflicts, integrating overlapping information, and creating new understanding that goes beyond individual sources.

## Examples

### Medical Diagnosis

```
SOURCES: Symptoms, Medical literature, Patient history
CONNECT: Symptoms match pneumonia, known exposure
SYNTHESIS: Begin treatment for pneumonia, test for COVID
```

## Capabilities

- Multi-source integration
- Connection identification
- Conflict resolution
- Insight generation
