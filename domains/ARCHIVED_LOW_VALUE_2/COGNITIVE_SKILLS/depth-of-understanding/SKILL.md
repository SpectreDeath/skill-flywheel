## Purpose

Build detailed logical justifications for conclusions through thorough understanding of mechanisms and implications.

## Description

Depth of understanding goes beyond surface knowledge to build comprehensive justifications: explaining how/why things work, what implications follow, and addressing counterarguments with evidence.

## Examples

### Algorithm Explanation

```
SURFACE: Binary search finds element in sorted array
MECHANISM: Divide search space in half each iteration
IMPLICATION: O(log n) complexity - 50,000x faster for 1M elements
JUSTIFICATION: Mathematical proof provided
```

## Capabilities

- Mechanism explanation
- Implication analysis
- Evidence-based justification
- Counterargument addressing
