## Purpose

Apply proven methods and standard procedures to solve problems efficiently rather than reinventing solutions.

## Description

Technique application involves identifying problem types, selecting appropriate standard techniques, applying them faithfully to procedures, and adapting when necessary for specific contexts.

## Examples

### Debugging Procedure

```
TYPE: HTTP 500 error → Server-side issue
TECHNIQUE: Divide and conquer debugging
STEPS: Logs → Database → Code → Config
RESULT: Firewall rule blocked port 5432
```

## Capabilities

- Problem type identification
- Technique selection
- Procedure application
- Adaptation for context
