# COGNITIVE_SKILLS

This domain contains skills for analytical thinking, problem-solving, and cognitive approaches derived from Vertical thinking (Edward de Bono) and Convergent thinking (Joy Paul Guilford).

## Skills

| Skill | Description |
|-------|-------------|
| [convergent-thinking](convergent-thinking/) | Main skill with 13 sub-skills overview |
| [logical-reasoning](logical-reasoning/) | Apply rational assessment to draw inferences |
| [sequential-problem-solving](sequential-problem-solving/) | Step-by-step problem resolution |
| [critical-evaluation](critical-evaluation/) | Assess quality with standards and probabilities |
| [analytical-thinking](analytical-thinking/) | Break down complex problems systematically |
| [decision-making](decision-making/) | Weigh alternatives and select best solution |
| [knowledge-synthesis](knowledge-synthesis/) | Combine knowledge from multiple sources |
| [information-retrieval](information-retrieval/) | Efficiently locate and apply information |
| [fact-based-assessment](fact-based-assessment/) | Evaluate using evidence not intuition |
| [accuracy-and-speed](accuracy-and-speed/) | Achieve correct answers under time pressure |
| [focused-selection](focused-selection/) | Filter irrelevant information |
| [depth-of-understanding](depth-of-understanding/) | Build detailed justifications |
| [conclusive-thinking](conclusive-thinking/) | Reach definitive unambiguous answers |
| [technique-application](technique-application/) | Apply proven methods and procedures |
