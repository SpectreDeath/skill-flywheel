# Logic Programming Skills

This directory contains a comprehensive collection of logic programming skills designed for AI agents and automated reasoning systems. These skills provide frameworks for constraint solving, formal methods, and logical reasoning across various domains including Prolog, SAT solving, constraint programming, temporal logic, and Datalog.

## Generated Skills

### 1. SKILL.prolog_knowledge_rep.md
**Purpose**: Prolog knowledge representation systems for AI applications, expert systems, and logical reasoning tasks.

**Key Capabilities**:
- Fact-based knowledge modeling and rule-based inference systems
- Semantic network representation and ontological structure design
- Constraint-based reasoning and knowledge base optimization
- Dynamic knowledge updates and consistency maintenance

**Applications**: Expert systems, knowledge bases, semantic reasoning, AI reasoning systems

### 2. SKILL.sat_solver_optimization.md
**Purpose**: SAT (Boolean Satisfiability) solver optimization for solving complex logical constraint problems.

**Key Capabilities**:
- Problem encoding optimization and solver selection/configuration
- Advanced heuristics (VSIDS, MOMS, activity-based) and clause learning
- Conflict analysis, parallel/incremental solving, performance profiling
- Integration with formal verification and planning systems

**Applications**: Formal verification, planning, scheduling, constraint satisfaction

### 3. SKILL.constraint_programming.md
**Purpose**: Constraint programming systems for combinatorial optimization problems.

**Key Capabilities**:
- Constraint modeling and domain propagation algorithms
- Advanced search strategies and global constraint implementation
- Optimization techniques and hybrid solving approaches
- Performance optimization and memory management

**Applications**: Scheduling, resource allocation, configuration, planning problems

### 4. SKILL.temporal_logic.md
**Purpose**: Temporal logic systems for reasoning about time-dependent properties and system behaviors.

**Key Capabilities**:
- LTL/CTL specification and model checking algorithms
- Temporal reasoning and real-time constraint handling
- Counterexample generation and temporal verification
- Integration with formal verification and runtime monitoring

**Applications**: Formal verification, reactive systems, real-time systems, protocol verification

### 5. SKILL.datalog_reasoning.md
**Purpose**: Datalog reasoning systems for knowledge base querying and recursive query processing.

**Key Capabilities**:
- Datalog program design and fixpoint computation
- Recursive query optimization (magic sets, semi-naive evaluation)
- Stratified negation handling and database integration
- Incremental evaluation and query optimization

**Applications**: Deductive databases, knowledge bases, program analysis, data analysis
