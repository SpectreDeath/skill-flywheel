# SkillsMP Ecosystem

Domain for SkillsMP (Skill Marketplace) integration - discover, learn from, orchestrate, and generate skills.

## Skills

| Skill | Description |
|-------|-------------|
| [skillsmp-master](skillsmp-master/) | Master skill for all SkillsMP operations |
| [skillsmp-api-client](skillsmp-api-client/) | HTTP client for SkillsMP API |
| [skillsmp-skill-learner](skillsmp-skill-learner/) | Learn patterns from SkillsMP skills |
| [skillsmp-skill-orchestrator](skillsmp-skill-orchestrator/) | Orchestrate skill assembly pipelines |
| [skillsmp-skill-factory](skillsmp-skill-factory/) | Generate skills from learned patterns |
| [skillsmp-skill-gap-discoverer](skillsmp-skill-gap-discoverer/) | Discover missing capabilities |

## Usage

Use these skills for:
- Searching AI agent skills from skillsmp.com
- Learning patterns from existing skills
- Building skill orchestration pipelines
- Generating new skills
- Finding capability gaps

**Requires**: API key from skillsmp.com
