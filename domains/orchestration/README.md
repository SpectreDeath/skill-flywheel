# Orchestration

Domain for skills related to skill orchestration, chaining, routing, and workflow management.

## Skills

| Skill | Description |
|-------|-------------|
| [task-skill-router](task-skill-router/) | Map user tasks to optimal skills |
| [skill-recommendation](skill-recommendation/) | Recommend relevant skills based on queries |
| [multi-skill-chaining-engine](multi-skill-chaining-engine/) | Chain multiple skills in sequence or parallel |

## Usage

Load these skills when building complex multi-skill workflows:
- Route user requests to appropriate skills
- Recommend skills for queries
- Orchestrate skill execution chains
