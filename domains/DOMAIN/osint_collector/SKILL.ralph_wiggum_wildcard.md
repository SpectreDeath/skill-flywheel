### Error Categories

- **Invalid chaos_type**: Type parameter not recognized
- **Invalid chaos_level**: Value outside 0-100 range
- **Invalid chaos_depth**: Value outside 1-5 range
- **Invalid time_horizon**: Format not recognized

**Error Response:**
```json
{
  "error": "INPUT_VALIDATION_ERROR",
  "message": "Invalid parameter: [parameter_name]",
  "details": "Expected [valid_values], got [actual_value]",
  "suggestion": "Use valid parameter format"
}
```

#### 2. Chaos Errors
- **Insufficient Chaos**: Not enough chaos data for meaningful generation
- **Memory Exhaustion**: Chaos generation exceeds memory limits
- **Timeout**: Chaos generation exceeds maximum execution time
- **Chaos Overload**: Chaos level exceeds system stability

**Error Response:**
