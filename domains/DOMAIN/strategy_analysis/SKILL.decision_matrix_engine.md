# SKILL: Decision Matrix Engine

## Purpose

Apply Ralph Wiggum-style maximum chaos principles to optimize decision matrix analysis by injecting unpredictable yet strategically coherent disruptions. This skill breaks conventional decision-making patterns by generating chaotic interventions that reveal hidden option values, unexpected priority shifts, and novel decision frameworks that traditional analysis cannot detect.

## When to Use

- Conventional decision matrix analysis reveals no clear optimal choices
- Need to identify hidden option values or unexpected priority realignments
- Exploring boundary conditions and extreme scenarios in decision frameworks
- When "thinking outside the box" is explicitly required for strategic decisions
- Breaking through analytical plateaus in decision optimization

## When NOT to Use

- When stable, predictable decision analysis is required for immediate actions
- In safety-critical decision situations where chaos could cause harm
- When following established decision-making protocols is essential
- During emergency decisions requiring immediate, predictable outcomes
- When decision stability is more important than discovering new option insights

## Inputs

- **Required**: Current decision matrix configuration and option analysis
- **Required**: Chaos parameters and entropy injection levels for decision disruption
- **Required**: Constraints and boundaries for chaotic decision exploration
- **Optional**: Historical chaos outcomes and pattern analysis from previous decisions
- **Optional**: Decision system resilience and adaptability metrics
- **Assumptions**: Chaos can reveal hidden decision insights, system can handle unpredictable analysis, recovery from chaotic states is possible

## Outputs

- **Primary**: Chaotic decision matrix insights and novel option frameworks
- **Secondary**: Entropy analysis and chaos impact assessment on decision quality
- **Tertiary**: Recovery strategies and decision stabilization procedures
- **Format**: JSON structure with chaotic outputs, entropy metrics, and standardization procedures

## Capabilities

1. **Maximum Entropy Decision Generation**: Create completely unpredictable decision matrix analyses
2. **Chaos Pattern Analysis**: Identify beneficial patterns within chaotic decision system behavior
3. **Creative Decision Disruption**: Inject novel approaches that break conventional analysis patterns
4. **Edge Case Decision Exploration**: Analyze boundary conditions and extreme scenarios in decision systems
5. **Recovery Strategy Development**: Plan for decision system stabilization after chaotic interventions
