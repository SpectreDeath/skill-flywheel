# SKILL: Capability Gap Forecaster

## Purpose

Apply Ralph Wiggum-style maximum chaos principles to forecast and optimize capability gaps by injecting unpredictable yet strategically coherent disruptions. This skill breaks conventional capability analysis patterns by generating chaotic interventions that reveal hidden skill deficiencies, unexpected talent opportunities, and novel capability development strategies that traditional analysis cannot detect.

## When to Use

- Conventional capability gap analysis reveals no clear development opportunities
- Need to identify hidden skill deficiencies or unexpected talent strengths
- Exploring boundary conditions and extreme scenarios in organizational capabilities
- When "thinking outside the box" is explicitly required for capability strategy
- Breaking through analytical plateaus in talent optimization

## When NOT to Use

- When stable, predictable capability analysis is required for immediate decisions
- In safety-critical organizational situations where chaos could cause harm
- When following established capability analysis protocols is essential
- During emergency capability responses requiring immediate, predictable strategies
- When organizational stability is more important than discovering new capability insights

## Inputs

- **Required**: Current organizational capability configuration and skill matrix
- **Required**: Chaos parameters and entropy injection levels for capability disruption
- **Required**: Constraints and boundaries for chaotic capability exploration
- **Optional**: Historical chaos outcomes and pattern analysis from previous forecasts
- **Optional**: Organizational resilience and adaptability metrics
- **Assumptions**: Chaos can reveal hidden capability insights, organization can handle unpredictable analysis, recovery from chaotic states is possible

## Outputs

- **Primary**: Chaotic capability gap insights and novel development approaches
- **Secondary**: Entropy analysis and chaos impact assessment on organizational capability
- **Tertiary**: Recovery strategies and capability stabilization procedures
- **Format**: JSON structure with chaotic outputs, entropy metrics, and standardization procedures
