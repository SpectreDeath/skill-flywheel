## Purpose

Reach definitive, unambiguous answers that resolve questions completely without remaining ambiguity.

## Description

Conclusive thinking involves assessing information sufficiency, evaluating evidence strength, determining if answers can be definitive, and stating conclusions with explicit confidence levels and caveats.

## Examples

### Root Cause Analysis

```
EVIDENCE: Memory peaked at 99.8%, deployment at 2:55pm
CONCLUSION: Memory leak in new deployment (HIGH confidence)
ACTION: Roll back, adjust auto-scaler threshold
```

## Capabilities

- Sufficiency assessment
- Evidence evaluation
- Definitive answer determination
- Ambiguity resolution
