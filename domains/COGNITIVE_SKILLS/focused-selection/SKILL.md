## Purpose

Filter out irrelevant information to work within the relevant frame of reference.

## Description

Focused selection involves defining the relevant scope/frame, identifying noise and distractions, filtering systematically, and verifying focus before proceeding. It prevents cognitive overload and maintains analytical clarity.

## Examples

### Requirements Analysis

```
FRAME: Customer dashboard, current quarter, real-time
RELEVANT: DAU, session duration, feature usage
IRRELEVANT: Employee surveys, 2020 data, competitor products
```

## Capabilities

- Frame definition
- Noise identification
- Systematic filtering
- Focus verification
