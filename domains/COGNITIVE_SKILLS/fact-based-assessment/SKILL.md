## Purpose

Evaluate situations and make decisions using observable evidence rather than intuition or assumptions.

## Description

Fact-based assessment involves gathering observable facts, separating them from opinions, verifying sources, analyzing patterns, and drawing evidence-based conclusions with explicit confidence levels.

## Examples

### Performance Assessment

```
FACTS: Velocity +15%, Bug count -60%, On-time +15%
OPINION: "Team is doing great" (deferred until analysis)
CONCLUSION: Evidence shows consistent improvement
```

## Capabilities

- Fact gathering
- Opinion separation
- Source verification
- Evidence analysis
