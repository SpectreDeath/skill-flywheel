## Purpose

Weigh alternatives systematically and select the best solution based on defined criteria using weighted decision matrices.

## Description

Decision-making involves defining the decision, identifying alternatives, establishing weighted criteria, evaluating options, and selecting the best solution. It provides a structured framework for making objective choices.

## Examples

### Technology Selection

```
OPTIONS: AWS, Google Cloud, Azure
CRITERIA: Cost 25%, Performance 25%, Features 20%
EVALUATION: AWS scores 4.2, GCP scores 4.2
DECISION: AWS selected (mature ecosystem)
```

## Capabilities

- Alternative identification
- Criteria weighting
- Decision matrix construction
- Risk assessment
