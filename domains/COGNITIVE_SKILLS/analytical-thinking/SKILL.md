## Purpose

Systematically break down complex problems into components to identify patterns and find optimal solutions.

## Description

Analytical thinking involves decomposition: breaking complex problems into sub-components, examining each, identifying patterns, and synthesizing insights to find optimal solutions. It emphasizes systematic approach and quantitative analysis.

## Examples

### Sales Decline Analysis

```
DECOMPOSE: Product, Region, Customer, Time, Channel
EXAMINE: Category A -5%, Region X -20%, New customers -30%
SYNTHESIZE: Root cause - marketing spend shift
CONCLUSION: Restore acquisition budget
```

## Capabilities

- Problem decomposition
- Pattern recognition
- Component analysis
- Root cause identification
