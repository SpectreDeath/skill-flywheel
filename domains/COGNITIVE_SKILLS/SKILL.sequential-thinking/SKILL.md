# Sequential Thinking Skills

A collection of skills for systematic reasoning and problem-solving.

## Skills in This Set

| Skill | Description |
|-------|-------------|
| [Chain of Thought Reasoning](chain-of-thought-reasoning.md) | Step-by-step logical reasoning for complex problems |
| [Problem Decomposition](problem-decomposition.md) | Break complex problems into manageable sub-problems |
| [Iterative Refinement](iterative-refinement.md) | Progressive improvement through feedback loops |
| [Root Cause Analysis](root-cause-analysis.md) | Systematically trace problems to fundamental causes |
| [Hypothesis Testing](hypothesis-testing.md) | Form and validate hypotheses with evidence |
| [Decision Tree Analysis](decision-tree-analysis.md) | Map decision paths to explore options and outcomes |

## When to Use Each Skill

| Situation | Recommended Skill |
|-----------|-------------------|
| Complex problem, unclear path | Chain of Thought |
| Overwhelming problem scope | Problem Decomposition |
| Solution not working well | Iterative Refinement |
| Recurring problem | Root Cause Analysis |
| Uncertain approach | Hypothesis Testing |
| Multiple options to evaluate | Decision Tree Analysis |

## Usage Pattern

```
1. Start with Problem Decomposition for large problems
2. Apply Chain of Thought for the sub-problems
3. Use Hypothesis Testing to validate approaches
4. Iterate with Iterative Refinement
5. Use Root Cause Analysis when issues recur
6. Apply Decision Tree Analysis for final choices
```
