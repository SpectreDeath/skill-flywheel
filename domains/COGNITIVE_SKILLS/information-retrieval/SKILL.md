## Purpose

Efficiently locate, access, and apply stored information to solve problems accurately.

## Description

Information retrieval is the skill of efficiently finding and applying relevant information from existing sources. It involves defining information needs, locating sources, executing searches, and verifying accuracy.

## Examples

### Documentation Search

```
NEED: Configure SSL in nginx
SEARCH: Internal docs → Official docs → Community
RESULT: nginx.org SSL configuration guide
```

## Capabilities

- Search strategy formulation
- Source location
- Query execution
- Information verification
