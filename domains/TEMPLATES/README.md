# Templates

Utility directory containing templates and helper scripts for skill development.

## Files

| File | Description |
|------|-------------|
| required_sections_templates.md | Templates for SKILL.md sections |
| batch_processor.py | Batch processor for adding required sections |
| compliance_checker.py | Compliance checking utility |
| enhanced_batch_processor.py | Enhanced batch processing |

## Usage

### required_sections_templates.md
Provides standardized templates for the four required sections in SKILL.md files:
- Purpose Section
- Examples Section  
- Implementation Notes Section
- Constraints Section

### Python Utilities
- `batch_processor.py` - Add required sections to skill files
- `compliance_checker.py` - Check skill file compliance
- `enhanced_batch_processor.py` - Enhanced batch processing capabilities

These utilities help maintain consistency across the skill repository.
