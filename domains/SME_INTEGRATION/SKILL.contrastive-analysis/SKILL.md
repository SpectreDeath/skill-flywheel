## Purpose

Performs contrastive analysis between texts to highlight stylistic differences and similarities.

## Description

The Contrastive Analysis module compares writing styles between texts to identify differences and similarities. It highlights distinctive features and common patterns.

2. **Feature Comparison**: Compare features
