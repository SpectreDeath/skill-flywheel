# SME Integration

Domain for integrating with the Semantic Memory Engine (SME) - a production-grade Model Context Protocol (MCP) Gateway for forensic AI capabilities.

## Overview

The SME (Lawnmower Man) provides a comprehensive suite of forensic tools accessible via MCP protocol. This domain exposes those capabilities as reusable skills for AI agent development.

## Architecture

The SME uses a dual-Python "Sidecar" architecture:
- **Operator**: Core logic (Python 3.14)
- **Sidecar**: AI bridge (Python 3.13)
- **Gateway**: MCP server exposing tools

## Hardware Requirements

Optimized for NVIDIA GeForce GTX 1660 Ti 6GB VRAM

## Quick Start

```python
# Connect to SME Gateway
from mcp import Client

client = Client("python gateway/mcp_server.py")
tools = client.list_tools()

# Execute a forensic tool
result = client.call_tool("trust_score_analysis", {"source": "https://example.com"})
```
